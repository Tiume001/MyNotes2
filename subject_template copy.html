Introduzione ai Puntatori

I puntatori sono variabili che contengono l'indirizzo di memoria di un'altra variabile. Sono fondamentali in C++ per la gestione dinamica della memoria.

Un puntatore non inizializzato può puntare a un'area di memoria casuale, causando errori gravi (Segmentation Fault).

Sintassi Base

int x = 10;
int *ptr = &x; // ptr contiene l'indirizzo di x

cout << *ptr; // Stampa 10 (dereferenziazione)

Allocazione Dinamica

Per allocare memoria nello heap si usa l'operatore new:

int *arr = new int[10]; // Array di 10 interi
// ... uso dell'array ...
delete[] arr; // Importante: liberare la memoria!
